# Security Incident Reporting System

## 1. Project Overview
This is an open-source web application designed for the Cyber Security and Digital Forensics Engineering program (Group 5). It contains a mandatory User Management Module for registration and authentication. Authorized users can record security incidents (e.g., Phishing, Malware, Data Breach), display all reported incident logs, and search for specific incidents using a unique Incident ID.

## 2. Installation Steps with Required Dependencies
1. Download and install **XAMPP** server environment.
2. Clone or extract this repository into your local server directory: `C:\xampp\htdocs\OpenSource_Assignment_CSDFE_Group_5`
3. Launch **XAMPP Control Panel** and activate **Apache** and **MySQL** services.
4. Access the portal via any browser using the link: `http://localhost/OpenSource_Assignment_CSDFE_Group_5/index.php`.
5. The underlying database structures will provision automatically during the first initialization via `db.php`.

## 3. Technologies Used
- **Backend Core:** PHP 8.2+
- **Database Engine:** MySQL 
- **Frontend Architecture:** HTML5 & CSS structures
- **Version Control:** Git & GitHub Distributed System
